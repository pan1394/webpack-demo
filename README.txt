一个简单的typescript + javascript + webpack 前端框架环境.

目录结构如下:
- webpack-demo
    |-asset目录 -> 资源目录.
    |-dummy -> 前端dummy数据.
    |-node_module -> 库文件目录.
    |-public目录 -> ts, js 打包后压缩文件, html 页面文件. 用于放置于web服务器.
    |-src 目录 -> ts, js 源代码目录.
    |-package.json -> npm 包管理文件.
    |-tsconfig.json -> typescript 编译配置文件.
    |-webpack.config.js -> webpack 打包配置文件.

用于快速搭建一个前端小型项目. 
比较重型的VUE, Angular对没有放置在里面.
